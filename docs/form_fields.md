# フォーム項目定義

## フォーム構成

- **日本語版**: 全15問（ふりがなあり）
- **その他5言語版**: 全14問（ふりがななし）
- 1フォームで言語を切り替え、条件分岐で表示制御

## v2 での改善点

1. **言語選択とフォームの2ページ構成** — `index.html` で言語を選び、`form.html` で選択言語のフォームを入力（条件分岐は動的表示切替）
2. 生年月日 → 入力しやすいUIに統合
3. パスポート写真 → Googleログイン不要
4. 約款 → フォーム内で内容確認可能に

- 職業
